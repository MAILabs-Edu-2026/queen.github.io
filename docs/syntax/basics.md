# Язык программирования

## Переменные и виды связывания

### Локальное связывание
Конструкция `let x = 42 in ...` создаёт локальное связывание переменной `x` со значением `42` в пределах указанного выражения.

**Пример:**
```bash
let x = 42 in x + 10 // Результат: 52
```



### Рекурсивное связывание
Конструкция `let rec fact = ... in ...` используется для определения рекурсивных функций (например, функции вычисления факториала). Ключевое слово `rec` позволяет функции ссылаться на саму себя внутри своего определения.

**Пример (вычисление факториала):**
```bash
let rec fact = \n ->
    if n <= 1 then 1
    else n * (fact (n - 1))
in fact 5 // Результат: 120
```

# Типы данных

В языке поддерживаются следующие типы данных:

* **`Int`** — целые числа.
  **Примеры:** `42`, `-5`, `1000`.

* **`Bool`** — булевы значения.
  **Допустимые значения:** `true`, `false`.

* **`String`** — строки. Записываются в двойных кавычках.
  **Примеры:** `"Hello"`, `"World"`, `"123"`.

* **`nil`, `cons`** — используются для представления списков:
    * `nil` — пустой список.

    * `cons(head, tail)` — конструктор для создания списка, где `head` — первый элемент, а `tail` — остальная часть списка.

        **Пример списка `[1; 2; 3]`:**

        `cons(1, cons(2, cons(3, nil)))`

* **`Tuple`** — кортежи. Упорядоченные коллекции фиксированного размера, которые могут содержать элементы разных типов.
  **Примеры:**
  `(1, "hello")`, `(true, 42)`.

* **`Var`** — переменные. Используются для хранения значений и ссылок на них.
  **Примеры:** `x`, `myVar`, `result`.

# Выражения

## Арифметические операции
Используются для работы с числовыми значениями (`Int`).

| Оператор | Операция | Пример | Результат |
|--------|----------|--------|-----------|
| `+`    | Сложение | `5 + 3` | `8` |
| `-`    | Вычитание | `10 - 4` | `6` |
| `*`    | Умножение | `3 * 7` | `21` |
| `/`    | Деление (целочисленное) | `15 / 3` | `5` |
| `%`    | Остаток от деления | `10 % 3` | `1` |

## Операции сравнения
Используются для сравнения значений. Возвращают булево значение (`Bool`).

| Оператор | Операция | Пример | Результат |
|--------|----------|--------|-----------|
| `==`   | Равно | `5 == 5` | `true` |
| `!=`   | Не равно | `5 != 3` | `true` |
| `<`    | Меньше | `2 < 4` | `true` |
| `>`    | Больше | `7 > 10` | `false` |
| `<=`   | Меньше или равно | `3 <= 3` | `true` |
| `>=`   | Больше или равно | `8 >= 9` | `false` |

## Логические операции
Работают с булевыми значениями (`Bool`).

| Оператор | Операция | Пример | Результат |
|--------|----------|--------|-----------|
| `&&`   | Логическое И (AND) | `true && false` | `false` |
| `\|\|`   | Логическое ИЛИ (OR) | `true \|\| false` | `true` |
| `!`    | Логическое НЕ (NOT) | `!true` | `false` |
