# Математические функции

1. `abs x` — модуль числа

Возвращает абсолютное значение числа (модуль) — положительное число без учёта знака.

**Синтаксис:** `abs x`.

2. `min x y` и `max x y` — минимум и максимум

* `min x y` возвращает меньшее из двух чисел.
* `max x y` возвращает большее из двух чисел.

**Синтаксис:**
* `min x y`;
* `max x y`.

3. `pow base exp` — возведение в степень

Возводит число в указанную степень.

**Синтаксис:** `pow base exp`, где:
* `base` — основание степени (число, которое возводится в степень);
* `exp` — показатель степени (в какую степень возводим).
