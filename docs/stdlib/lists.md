# Списки

Списки строятся из двух базовых конструкций:
* `nil` — пустой список;
* `cons` — конструктор для добавления элемента в начало списка (создаёт пару «элемент + хвост списка»).


## Основные функции для работы со списками

1. **`range min max`** — создаёт список чисел в заданном диапазоне.
   * `min` — начальное значение диапазона;
   * `max` — конечное значение диапазона (включительно).

2. **`map f lst`** — применяет функцию `f` к каждому элементу списка `lst`, возвращая новый список с результатами.
   * `f` — функция, которая будет применена к каждому элементу;
   * `lst` — исходный список.

3. **`filter f lst`** — фильтрует элементы списка `lst` по заданному предикату `f`. Возвращает список, содержащий только те элементы, для которых `f` вернула `true`.
   * `f` — предикат (функция, возвращающая `Bool`);
   * `lst` — исходный список.

4. **`fold f acc lst`** — выполняет свёртку списка `lst` с использованием функции `f` и начального значения `acc`. Позволяет агрегировать элементы списка в одно значение (сумма, произведение и т. д.).
   * `f` — функция свёртки (принимает аккумулятор и текущий элемент, возвращает новое значение аккумулятора);
   * `acc` — начальное значение аккумулятора;
   * `lst` — список, который нужно свернуть.

5. **`append lst1 lst2`** — объединяет два списка `lst1` и `lst2` в один, возвращая новый список. Элементы `lst2` добавляются в конец `lst1`.
   * `lst1` — первый список;
   * `lst2` — второй список.

## Сводная таблица функций

| Функция | Синтаксис | Назначение |
|-------|---------|---------|
| Диапазон | `range min max` | Создание списка чисел от `min` до `max` включительно |
| Отображение | `map f lst` | Применение функции `f` ко всем элементам списка `lst` |
| Фильтрация | `filter f lst` | Оставление в списке `lst` только элементов, удовлетворяющих предикату `f` |
| Свёртка | `fold f acc lst` | Агрегация элементов списка `lst` в одно значение с начальным аккумулятором `acc` |
| Конкатенация | `append lst1 lst2` | Объединение списков `lst1` и `lst2` в один |
