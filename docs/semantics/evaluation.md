# Ленивая оценка

Queen вычисляет выражения только тогда, когда их значение нужно.

**Thunk**

`Thunk` хранит выражение и окружение. При первом обращении он вычисляется и запоминает результат.

```queen
let rec loop = \x -> loop x in
let always42 = \x -> 42 in
always42 (loop 0)
```

Результат:

```text
42
```

Аргумент `loop 0` не вычисляется, потому что функция его не использует.

**Где значение форсируется**

* в условии `if`;
* при `match`;
* в арифметике и сравнениях;
* в `head`, `tail`, `fst`, `snd`;
* в `print`, `readFile`, `writeFile`.
