# Команды CLI для Queen


## Основные команды

* **`queen run <file.qx>`** — запуск скрипта.
  Выполняет скрипт, указанный в файле с расширением `.qx`.

* **`queen help`** — отображение справочной информации по CLI.
  Выводит список доступных команд и краткое описание их функционала.
