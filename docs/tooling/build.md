# Руководство по сборке и тестированию проекта

## Требования

Для работы проекта необходимы следующие компоненты:

* **.NET SDK версии 10.0 или выше** — платформа для разработки, сборки и запуска приложения.
* **FParsec 1.1.1** — библиотека для парсинга, используемая в проекте.

## Основные команды .NET CLI


### 1. Сборка проекта

Команда для компиляции проекта и проверки кода на ошибки:

```bash
dotnet build queen.slnx
```
## Что делает:

- Компилирует исходный код F# в исполняемые файлы.
- Автоматически проверяет и восстанавливает зависимости (например, FParsec 1.1.1).
- Выводит подробные сообщения об ошибках компиляции, если они есть.

### 2. Запуск примера

**Команда:**
```bash
dotnet run --project src/queen/queen.fsproj -- run examples/factorial.qx
```

### Что делает:

- Запускает интерпретатор Queen.
- Выполняет пример `examples/factorial.qx`.
- Показывает результат работы программы.

### 3. Упаковка проекта

**Команда:**
```bash
dotnet pack src/queen/queen.fsproj -c Release
```

### Что делает:

- Собирает проект и упаковывает его в формат NuGet-пакета (.nupkg).
- Включает метаданные из `src/queen/queen.fsproj` (версия, авторы, target framework).
- Подготавливает пакет к публикации в NuGet-репозитории или локальному использованию.
