# Установка и первый запуск языка Queen

Чтобы начать работу с языком программирования Queen, выполните следующие шаги:

## 1. Проверьте наличие .NET SDK

Убедитесь, что у вас установлен **.NET SDK версии 10.0 или новее**.

Если SDK отсутствует или устарел, скачайте последнюю версию с официального сайта:
🔗 [https://dotnet.microsoft.com/download](https://dotnet.microsoft.com/download)

Проверьте установку:
```bash
dotnet --version
```

## 2. Клонируйте репозиторий


Откройте терминал (командную строку) и выполните команду для клонирования репозитория:

```bash
git clone https://github.com/MAILabs-Edu-2026/funcpro-coursework-queen-anne-s-revenge
```

## 3. Перейдите в директорию проекта


После клонирования репозитория перейдите в папку проекта с помощью команды:

```bash
cd funcpro-coursework-queen-anne-s-revenge/
```

## 4. Соберите проект


Выполните сборку проекта с помощью команды:
```bash
dotnet build queen.slnx
```

Эта команда скомпилирует исходные файлы и подготовит среду для запуска CLI.

## 5. Запуск скриптов

Вы можете запускать скрипты напрямую из исходников:

```bash
dotnet run --project src/queen/queen.fsproj -- run examples/factorial.qx
```

Для проверки доступных команд:

```bash
dotnet run --project src/queen/queen.fsproj -- help
```

## 6. Установка как .NET Tool (опционально)

Так как проект написан на .NET, он одинаково работает на Windows, macOS и Linux. Чтобы пользоваться командой `queen` глобально из любой папки (без `dotnet run`), упакуйте и установите инструмент:

```bash
dotnet pack queen.slnx
dotnet tool install --global --add-source ./nupkg queen
```

Теперь вы можете использовать компилятор короткой командой:

```bash
queen run examples/factorial.qx
queen help
```

## 7. Создайте и запустите первый скрипт


**Шаг 1.** Создайте файл с расширением `.qx` (например, `hello.qx`) в любом текстовом редакторе.

**Шаг 2.** Добавьте в файл следующую строку кода:

```queen
"Hello, Queen!"
```

**Шаг 3.** Запустите скрипт с помощью CLI:

```bash
queen run hello.qx
```

После выполнения этой команды в консоли появится вывод:

```bash
Hello, Queen!
```


## 8. Справка CLI

Чтобы получить справку по командам, выполните:

```bash
queen help
```


Эта команда выведет список доступных команд.

Теперь вы готовы к работе с языком Queen!
